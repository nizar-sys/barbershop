USE API FOR ACCESS DATABASE

base path : http://api-data.unaux.com

Endpoint List User in Database : user/list.php
Endpoint Detail User in Database : user/detail.php?uid=uid (example: ?uid=3)

Endpoint List Input Booking in Database : booking/list.php
Endpoint Detail Input Booking in Database : booking/detail.php?id=4 (example: ?id=4)
